Task C – Multi-label, multi-task video classification (SOA)

Challenge 2018 → Task C – Multi-label, multi-task video classification (SOA)

SOA (Scenes-Objects-Actions) is a multi-label and multi-task dataset that includes video clips of about 10 seconds. Each video has multiple annotations for Scene, Object and Action. The rich annotations from SOA will help us to better understand video data and explore the correlations among scene, object and action.

For information related to this task, please contact: hengwang@fb.com, mdf@fb.com

Dataset

The SOA dataset will be used for this challenge. Unlike conventional video datasets, videos from SOA are uniformly sampled from Internet users, as we intend to avoid any bias introduced by querying search engines. Trained annotators are asked to provide labels that can best describe the video in three different aspects, i.e., scene, object and action. These raw labels are then merged, split and renamed to generate a final taxonomy for SOA. All the annotations are then verified once more by the trained annotators for agreement with the taxonomy. In the end, SOA has about 500k videos with 2.5M labels covering about 50 scene classes, 350 object classes, and 150 action classes. The dataset is randomly split into training, validation and testing set with a percentage of 70, 10 and 20 respectively.

Evaluation Metric

Performance on SOA is measured by computing the average precision (AP) for each class since it is a multi-label dataset. For each individual task (e.g., Scene), we report the mean AP over all its classes (mAP). For the multi task case (e.g., SOA), we weight each task to obtain mAP_SOA = 1/6*mAP_Scene + 1/3*mAP_Object + 1/2*mAP_Action, to approximate the relevance of each task to video understanding. The final ranking in the leaderboard will be based on mAP_SOA, and we will include the mAP for each individual task as well.

Submission Format

Please follow the following CSV format when submitting your results for the challenge: 

            video_id,labels,scores
            v1_00000000,"O_hat,A_human_singing,S_grass_field","0.65,0.15,0.08"
            v1_00000001,"O_car,O_shoe,A_speaking_to_camera","0.75,0.25,0.18"

The example above is illustrative. For each video, you can provide at most 10 predictions. The labels and scores must be in the same order.
